## 安装

安装环境推荐基于3.0.0版本以上的【[**VeighNa Studio**](https://www.vnpy.com)】。

下载源代码后，解压后在cmd中运行：

```
pip install .
```

## 使用

在 C:\\Users\\xxx.vntrader\vt\_setting.json vnpy 配置文件中修改：

```json
{"datafeed.name": "adata"}
```
